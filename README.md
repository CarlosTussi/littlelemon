# littlelemon
Project: Restaurant Website

To run:
1) Clone repository
```
git clone https://github.com/CarlosTussi/littlelemon.git
```
2) On the same directory as "package.json" run the command
```
npm install
```

3) Execute 
```
npm start
```

4) Inside the brownser navigate to http://localhost:3000/
